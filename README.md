# SoSaltix Linux

Arch-based Linux distribution with KDE Plasma

## Features
- Based on Arch Linux
- KDE Plasma desktop
- Fastfetch with custom ASCII logo
- Optimized for performance

## Build instructions
```bash
git clone https://github.com/ktoto42-oss/sosaltix
cd sosaltix
sudo mkarchiso -v -w work/ -o out/ sosaltix-profile/

Requirements
archiso package
Root privileges
~10GB free space

License
MIT
EOF
