# Sosaltix Linux

Arch-based Linux distribution

## Features
- Based on Arch Linux
- Fastfetch with custom ASCII logo
- Optimized for performance

## Build instructions
git clone https://github.com/ktoto42-oss/sosaltix
cd sosaltix
sudo mkarchiso -v -w work/ -o out/ sosaltix-profile/


Requirements
archiso package
Root privileges
~10GB free space
